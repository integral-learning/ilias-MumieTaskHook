## Code Style

This project uses [PHP-CS-Fixer](https://github.com/PHP-CS-Fixer/PHP-CS-Fixer) to enforce consistent code style. Run the fixer locally before committing.

**Setup:**
```bash
composer install
```

**Run:**
```bash
./vendor/bin/php-cs-fixer fix
```
